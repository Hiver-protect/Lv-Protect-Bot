# Bot Protect Lv

Bot Discord professionnel multi-serveur (Node.js, discord.js)

## Fonctionnalités principales
- Protection anti-raid, anti-spam, anti-bot
- Système de tickets avancé
- Vérification utilisateurs (captcha, comptes récents...)
- Attribution automatique de rôles
- Commandes de gestion (ban, kick, mute, unmute, warn, purge)
- Configuration par serveur (/config)
- Commandes slash globales
- Logs de sécurité

## Installation
1. Clonez le dépôt :
	```bash
	git clone https://github.com/votre-utilisateur/bot-protect-lv.git
	cd bot-protect-lv
	```
2. Installez les dépendances :
	```bash
	npm install
	```
3. Copiez `.env.example` en `.env` et remplissez les variables nécessaires.
4. Lancez le bot :
	```bash
	npm start
	```

## Configuration
- Utilisez la commande `/config` sur chaque serveur pour définir les salons, rôles, etc.
- Les commandes slash sont globales et disponibles partout où le bot est invité.

## Déploiement des commandes
- Pour (re)déployer les commandes slash globales, exécutez le module `deploy-commands` après avoir modifié les commandes.
